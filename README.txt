Compilação:
 Após acessar a pasta referente ao trabalho, digite o seguinte comando no terminal:
    make main

Execução: 
 Após a compilação ter sido realizada, basta digitar o seguinte comando no terminal:
    ./main

Bibliotecas utilizadas:

    <pthread.h>: o cabeçalho pthread.h define constantes simbólicas, define expressões contantes de tempo de compilação válidas como inicializadoras para alguns tipos de dados, define tipos de dados descritos na biblioteca textbfsys/types.h e define funções pthread.h.

    <unistd.h>: define constantes e tipos simbólicos, além de declarar diversas funções.

    <time.h>: defines tipos de tempos.
